簡介

緣起

再者，強者我同學以臺灣經濟為念，力薦必須製作勝文滑出來以促進螢幕汰換，進而推動臺灣經濟發展，更加堅定了我寫勝文滑出來的決心！
身為不靠家族的碼農第一代，一直玩一直玩，我們責無旁貸！


資料來源

不同於「柯 P 滑出來對你說政見」使用「柯文哲野生官網 unlimited」所提供之 API 服務

因此並無法透過 API 取得勝文的證件政見，API 真 是 太 危 險 了 ， 怎 麼 可 以 公 開 呢 ！ ？
（什麼！？你說你不知道 API 是什麼？沒關係，讓維基百科告訴你 API 是什麼）
還好透過無敵黨證與Line群組，我們最終成功地獨家取得勝文的政見！

透過「勝文滑出來對你說政見」，讓您可以簡單、輕鬆地在 個人網站 或 網誌 中埋入「勝文彩蛋」
並讓更多人得以了解連勝文之政見。更多相關說明，請拜訪 連勝文的競選官方網站


在安裝「勝文滑出來對你說政見」之後 ... 我驚呆了

當您在的網頁中安裝了「勝文滑出來對你說政見」Plugin 之後，神奇的事情發生了
到訪者進入您的頁面、並捲動到頁面底端之後，勝文就會   滑 出 來 向 他 們 說 明 政 見  
每次的瀏覽，Plugin 都會透過不具名黨政高層，從 連勝文官方網站 中隨機的取得一筆政見，來對瀏覽者說明。

範例

預設效果   從左邊滑出來   快一點   非常非常慢   跳出來   鬼鬼祟祟  

如何安裝

步驟 1 : 請先在您的網頁中，引用 jquery，如果已經有了，可省略此步驟

<script src="//code.jquery.com/jquery-1.11.1.min.js"></script>

步驟 2 : 請在您的網頁中 </head> 之前，引用下方 JS 和 CSS

<link rel="stylesheet" href="//qcl.github.io/lien/src/lientalks.css"></link>
<script src="//qcl.github.io/lien/src/lientalks.js"></script>
<script>
  $(function(){
      $.lientalks();
  });
</script>

參數設定

Option Value
api API 來源網址，可是勝文沒有提供 API。（等等，沒有 API 那這組參數是幹什麼用的？）
預設：null
image 圖片網址，可設定 單張 或 多張圖片，設定多張圖片請用 array 包住
預設：
['http://qcl.github.io/lien/img/lien.png',
'http://qcl.github.io/lien/img/lien-1.png',
'http://qcl.github.io/lien/img/lien-2.png']
height 圖片高度
預設：450
width 圖片寬度
預設：466
effect 連公子出場效果，可選參數：default, fast, slow, veryslow, jump, sneaky
預設：default
popup_effect 對話框出現效果，可選參數：default, fade, slide, zoom
預設：fade
popup_radius 對話框圓角
預設：8px
popup_color 對話框文字顏色
預設：black
popup_bgcolor 對話框背景顏色
預設：beige
readmore_color 了解更多連結顏色
預設：brown
comein_position 設定連公子出場的位置時機，數值範圍：0 ~ 100，以頁面捲動的百分比計算
例如設定為 50 ，表示捲動至頁面高度之 50% 後，顯示連公子
預設：80
default_text 對話框預設文字
預設：你好，我是連戰的兒子連勝文
enter_from 設定連公子從左側或右側出場，可選參數：left,right
預設：right
enter_distance 設定連公子出場後，與邊界的距離
預設：-130

使用範例

CODE

  $.lientalks({
    image: [
      '../img/37.png'
    ],
    width: 800,
    height: 774,
    effect: 'jump',
    popup_effect: 'zoom',
    default_text: '你好，我是連勝文競選總部總幹事，蔡正元'
  });
